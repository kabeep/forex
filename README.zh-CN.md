## 📖 简介

> `forex` 全称是 `Foreign Exchange`，它的目的不是用于 `FX` 外汇交易，请慎重用于投资相关决策。

在浏览器或终端中获取每日汇率，免费且无速率限制。

查看[文档][docs-url]。
